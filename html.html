Квантовая химия

Программа готовит специалистов в области моделирования молекулярных структур с использованием квантово-механических подходов и виртуальной реальности.

Основные дисциплины:

  • Квантовая механика и химия
  • Вычислительные методы в химии
  • Молекулярное моделирование
  • Программирование для химиков (Python, C++)
  • Виртуальные лаборатории

Карьерные перспективы:

  • Разработчик молекулярных моделей
  • Специалист по компьютерному дизайну лекарств
  • Исследователь в области материаловедения
  • Научный сотрудник в R&D центрах

Студенты работают с современным оборудованием:

  • VR-лаборатории с полным погружением
  • Квантовые симуляторы
  • Высокопроизводительные вычислительные кластеры

Нейротехнологии

Инновационная программа на стыке химии, биологии и искусственного интеллекта, готовящая специалистов для работы с нейронными сетями в химической промышленности.

Основные дисциплины:

  • Основы нейронаук
  • Машинное обучение для химиков
  • Анализ больших данных в химии
  • Биоинформатика
  • Оптимизация химических процессов с помощью ИИ

Практическое применение:

  • Прогнозирование свойств новых материалов
  • Автоматизация химического синтеза
  • Оптимизация промышленных процессов
  • Разработка "умных" катализаторов

Студенты участвуют в реальных проектах:

  • Разработка нейросетей для анализа спектров
  • Создание систем предсказания биологической активности
  • Оптимизация технологических процессов на предприятиях

Космическая химия

Уникальная программа, готовящая специалистов для разработки материалов и химических систем, способных функционировать в экстремальных условиях космоса.

Основные дисциплины:

  • Химия экстремальных состояний
  • Материалы для космических аппаратов
  • Системы жизнеобеспечения в космосе
  • Радиационная химия
  • Космическое материаловедение

Направления исследований:

  • Термостойкие полимеры и композиты
  • Радиационно-защитные материалы
  • Системы регенерации воздуха и воды
  • Топлива для космических аппаратов
  • Химические источники энергии

Партнеры программы:

  • Роскосмос
  • НПО Энергомаш
  • Центр подготовки космонавтов
  • Пермские предприятия аэрокосмической отрасли
